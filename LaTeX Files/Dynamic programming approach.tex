Figure 1 showcases the possibilities of bus travelling from stop 2. It can travel to stops 3,4,5 or 6. The equations representing each of these possibilities are represented towards its right. The aim is to find minimum value out of each of these equations. However each of the equation contains the function $g$ and hence it needs to be solved. The figures and equations below represent the further solution of the equations until an integer value is found for the function $g$.

So now from stop 6 the bus moves back to the starting point, stop 2. Same logic is used for all the equations. The general formula for this is :
\begin{equation}
    g(j,\phi) = w(j,i)
\end{equation}
